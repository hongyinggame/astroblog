&emsp;&emsp;插入单行数据命令：

```
INSERT INTO 表名(字段名1,字段名2...) VALUES(值1,值2...);
```
&emsp;&emsp;插入多行数据命令：

```
INSERT INTO 表名(字段名1,字段名2...) VALUES(a1,a2...),(b1,b2...),...;
```
